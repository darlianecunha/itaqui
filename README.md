Dashboard Sustentavel do Porto do Itaqui - Comprehensive operational and sustainability indicators for one of Brazil's most important deepwater ports.

The Port of Itaqui, located in Sao Luis, Maranhao, is a strategic maritime hub handling grain exports, mineral products, and liquid bulk cargo. This dashboard provides real-time and historical data on port operations and sustainability metrics.

## About Port of Itaqui

The Port of Itaqui is one of the largest and most modern deepwater ports in Brazil. Key characteristics:

- **Location**: Sao Luis, Maranhao State
- **Status**: One of Brazil's busiest and most important ports
- **Cargo Focus**: Grain, minerals, and liquid bulk
- **Strategic Importance**: Critical infrastructure for Brazilian exports

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The dashboard will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

### Deployment

The project is configured for deployment on Vercel. Push to the main branch to trigger automatic deployment.
